SubQuery Network API

# EraManager

Produce epochs based on a period to coordinate contracts. Staking and reward distributing are running based on Eras

# STATES

# settings

ISettings contract which stores SubQuery network contracts address

contract ISettings settings

# eraPeriod

Era period in second

uint256 eraPeriod

# eraNumber

Current Era number

uint256 eraNumber

# eraStartTime

Current era start time in unix timestamp

uint256 eraStartTime

# maintenance

Maintenance mode, when maintenance mode, only owner can update

bool maintenance

# EVENTS

# EraPeriodUpdate

Emitted when admin update the eraPeriod

event EraPeriodUpdate(uint256 era, uint256 eraPeriod)

# NewEraStart

Emitted when new Era started

event NewEraStart(uint256 era, address caller)
