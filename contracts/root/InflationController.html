# InflationController

The InflationController contract mint the inflation SQT token to a set address at a set inflation rate. It also provide the manual way to mint SQT Token to admin.

# settings

contract ISettings settings

# inflationRate

The one year inflation rate for SQT token

uint256 inflationRate

# inflationDestination

The address to recevie the inflation SQT token

address inflationDestination

# lastInflationTimestamp

Last inflation timestamp

uint256 lastInflationTimestamp

# YEAR_SECONDS

Seconds for the Julian year

uint256 YEAR_SECONDS

# InflationStart

Emitted when inflation started

event InflationStart()

# FUNCTIONS

# initialize

Initialize the contract to setup parameters: inflationRate, inflationDestination, lastInflationTimestamp

function initialize(contract ISettings _settings, uint256 _inflationRate, address _inflationDestination) external
Name Type Description
_settings contract ISettings SQT token address
_inflationRate uint256 One year inflationRate for SQT token
_inflationDestination address Address to receive the inflation SQT token

# setSettings

Update setting state.

function setSettings(contract ISettings _settings) external
Name Type Description
_settings contract ISettings ISettings contract

# setInflationRate

Set the inflation rate

function setInflationRate(uint256 _inflationRate) external
Name Type Description
_inflationRate uint256 One year inflationRate for SQT token

# setInflationDestination

Set the address to receive the inflation SQT token

function setInflationDestination(address _inflationDestination) external
Name Type Description
_inflationDestination address Address to receive the inflation SQT token

# mintInflatedTokens

Can only called by eraManager when startNewEra, it will calculate and mint the inflation SQT token for last Era according to the inflation rate.

function mintInflatedTokens() external

# mintSQT

Can only called by admin account to mint the specified amount of SQT token.

function mintSQT(address _destination, uint256 _amount) external
Name Type Description
_destination address Address to receive the minted SQT token
_amount uint256 amount SQT token to mint
